# INF1900

## Liste des ports (0-7)
### PORTA
A0-A7 : Display (0-7)  

### PORTB
B3&B5 : Son (OC0A)  
B4&B6 : Sonar  
B4-B7 : Programmation  

### PORTC
C0-C1 : Memoire (I2C)  
C2-C7 : Capteur de ligne  

### PORTD
D0-D1 : RS-232  
D2 : Bouton (INT0)  
D4-D7 : Direction pour les roues (PWM sur D4, D5)
